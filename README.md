# Logistic Company Landing Page

A responsive and modern **landing page** for a logistic company built with **React** and **Tailwind CSS**. This project showcases the company's services, fleet, tracking options, and client testimonials with a clean and professional design.

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices.
- **Hero Section**: Catchy headline with call-to-action buttons.
- **Services Section**: Highlight logistic services like shipping, fleet, tracking, and delivery.
- **About Section**: Brief information about the company.
- **Testimonials/Clients Section**: Showcases client feedback and logos.
- **Footer**: Contact information and social media links.
- **Modern UI**: Built using Tailwind CSS with smooth transitions and hover effects.

## Technologies Used

- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- HTML5 & CSS3
- JavaScript (ES6+)
